2. What is the difference between local storage and session storage?

Ans: In simple word local storage and session storage are almost same. Yes, all most same but there is a catch. They have the following rules:

• Store key pairs for a particular domain.

• Only use strings to stored values

• Around 5MB of storage space limit

The catch or one major difference between localStorage and sessionStorage is the expiration date.
Local Storage: the storage across browser sessions. Now if the tab is opened in a new domain, or the browser is closed & re-opened, the data is saved. No data expiration.
Session Storage: when the page session ends, session lasts as long as the tab or the browser is open. So, in simple word when we open a new tab or a new window, this creates a new session and new storage.
